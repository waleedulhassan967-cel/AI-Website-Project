Project Description
This is a modern, interactive web application built to help students generate funny excuses.
Tool & Prompt Details
AI Tool Used: Emergent Agent & ChatGPT
Primary Prompt: "Create a modern, interactive Excuse Generator for Students web application using HTML, CSS, and JavaScript only. The website should generate random student excuses in Urdu language only."
Key Features
Catchy Hero Section with quirky typography.
Animated 'Generate Excuse' button.
Pastel color palette (Mint Green and Sunny Yellow).
